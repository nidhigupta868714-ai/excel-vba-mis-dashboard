# 📊 Dynamic MIS Performance Dashboard (Excel + VBA)

## 📌 Project Overview

This project is an interactive **Management Information System (MIS) Dashboard** built using Microsoft Excel with advanced features like **Macros and VBA automation**.

It is designed to analyze and monitor:

* Employee performance
* Efficiency metrics
* Break time tracking
* Overall productivity insights

The dashboard transforms raw data into meaningful insights through dynamic visualizations and automated calculations.

---

## 🚀 Key Features

* 📊 Interactive Dashboard with slicers and filters
* ⚙️ Automated data processing using VBA
* 📈 Performance & efficiency tracking
* 🔄 Dynamic data updates (no manual calculations required)
* 🧩 Multi-sheet structured workflow (Basic → Efficiency → Summary → Dashboard)

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* VBA (Visual Basic for Applications)
* Pivot Tables & Charts
* Dynamic Named Ranges

## 📁 Project Structure

* **Basic** → Raw data input
* **Break** → Break time tracking
* **Efficiency** → Calculations & metrics
* **Overall Performance** → Aggregated analysis
* **Summary** → Final insights
* **Dashboard** → Interactive visualization layer

---

## ⚙️ How to Use

1. Download the `.xlsm` file
2. Open in Microsoft Excel
3. Enable Macros when prompted
4. Navigate to the **Dashboard** sheet
5. Use slicers/filters to explore insights
